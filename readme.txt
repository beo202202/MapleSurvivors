--요구 환경--
파이썬 3.8.6

pygame이 3.11을 아직 지원 안하기 때문에 어쩔 수 없이 3.8.6 없는 선택입니다.

--계획--
- 시간에 따라 스테이지, 몬스터 리스트를 구현
- 몬스터를 죽이면 경험치를 얻고 레벨업 구현
- 화면에 exp 게이지 구현

--화면--
구현:
- 화면에 분 초 카운트

--소리--
구현:
배경 소리, 스킬 소리

예정:
스테이지에 따라 배경 소리
피격음
스킬에 따라 스킬음

--스킬--
- 이미지 구현 (FPS수에 따라 자동으로 조절됨.)
- 데미지 구현 (데미지-= 몬스터 방어력)

--몬스터--
- 초기 레벨위주로 몬스터 추가
예정:
- FPS에 따라 유동적으로 변경
- 움직일 시 모션
- 죽을 시 모션

--치명적인 문제--
- 몹이 많아지면 몹 이동속도 및 FPS가 떨어지는 것 같음.

--발견된 문제--
- 겹치는 문제를 해결하는 도중 간혹 0으로 나누는 문제가 생긴다. >>> 기능 보류
- 몬스터 생성 시 북쪽과 서쪽을 위주로 생성된다. >>> 추후 수정
